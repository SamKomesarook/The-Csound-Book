Instrument Control: Time Reading

clock

  kr      clock

Description

Return the value of the "wall-clock" time as provided by the operating system, in seconds. Since resolution is operating system dependant, it may be coarser or finer than seconds. On GNU/Linux systems it resolves to microseconds. On Windows, resolution is to seconds.
