Utilities: Analysis File Generation

hetro -sr n infilename outfilename Hetrodyne analysis sample rate
  -c n infilename outfilename Hetrodyne analysis channel number
  -b n infilename outfilename Hetrodyne analysis segment begin time
  -d n infilename outfilename Hetrodyne analysis segment duration
  -f n infilename outfilename Hetrodyne analysis beginning frequency
  -h n infilename outfilename Hetrodyne analysis number of partials
  -M n infilename outfilename Hetrodyne analysis maximum amplitude
  -m n infilename outfilename Hetrodyne analysis minimum amplitude
  -n n infilename outfilename Hetrodyne analysis number of breakpoints
  -l n infilename outfilename Hetrodyne analysis use third order low-pass filter with fc of n
 
lpanal -a infilename outfilename LPC analysis write filter pole instead of coeffecients
  -s n infilename outfilename LPC analysis sample rate
  -c n infilename outfilename LPC analysis channel number
  -b n infilename outfilename LPC analysis segment begin time
  -d n infilename outfilename LPC analysis segment duration
  -p n infilename outfilename LPC analysis number of poles
  -h n infilename outfilename LPC analysis hop size in samples
  -C s infilename outfilename LPC analysis text string for comments
  -P n infilename outfilename LPC analysis lowest frequency
  -Q n infilename outfilename LPC analysis highest frequency
  -v n infilename outfilename LPC analysis verbosity level of terminal messages
         
 
pvanal -s n infilename outfilename STFT analysis sample rate
  -c n infilename outfilename STFT analysis channel number
  -b n infilename outfilename STFT analysis segment begin time
  -d n infilename outfilename STFT analysis segment duration
  -n n infilename outfilename STFT analysis frame size
  -w n infilename outfilename STFT analysis window overlap factor
  -h n infilename outfilename STFT analysis hop size in samples

Utilities: File Queries

sndinfo soundfilename get info about one or more sound files soundfilename
 
pvlook -bb n infilename   STFT analysis file formatted text output beginning bin number
  -eb n infilename   STFT analysis file formatted text output ending bin number
  -bf n infilename   STFT analysis file formatted text output beginning frame number
  -ef n infilename   STFT analysis file formatted text output ending frame number
  -i infilename   STFT analysis file formatted text output as integers

Utilities: File Conversions

sdif2ads   infilename outfilename   Convert files Sound Description Interchange Format (SDIF) to the format usable by Csound's adsyn opcode.
 
dnoise -N num -i noisefilename -o outputfilename inputfilename number of bandpass filters
  -w num -i noisefilename -o outputfilename inputfilename filter overlap factor: 0, 1, (2), or 3.
  -M num -i noisefilename -o outputfilename inputfilename analysis window length
  -L num -i noisefilename -o outputfilename inputfilename synthesis window length
  -D num -i noisefilename -o outputfilename inputfilename decimation factor
  -b time -i noisefilename -o outputfilename inputfilename begin time in noise reference soundfile
  -B num -i noisefilename -o outputfilename inputfilename starting sample in noise reference soundfile
  -e time -i noisefilename -o outputfilename inputfilename end time in noise reference soundfile
  -E sampnum -i noisefilename -o outputfilename inputfilename final sample in noise reference soundfile
  -t num -i noisefilename -o outputfilename inputfilename threshold above noise reference in dB
  -S num -i noisefilename -o outputfilename inputfilename sharpness of noise-gate turnoff in the range 1 to 5
  -n num -i noisefilename -o outputfilename inputfilename number of FFT frames to average
  -m num -i noisefilename -o outputfilename inputfilename minimum gain of noise-gate when off, in dB
  -A -i noisefilename -o outputfilename inputfilename AIFF format output
  -W -i noisefilename -o outputfilename inputfilename WAV format output
  -J -i noisefilename -o outputfilename inputfilename IRCAM format output
  -h -i noisefilename -o outputfilename inputfilename skip soundfile header
  -8 -i noisefilename -o outputfilename inputfilename 8-bit unsigned character sound samples
  -c -i noisefilename -o outputfilename inputfilename 8-bit signed character sound samples
  -a -i noisefilename -o outputfilename inputfilename alaw sound samples
  -u -i noisefilename -o outputfilename inputfilename ulaw sound samples
  -s -i noisefilename -o outputfilename inputfilename short integer sound samples
  -l -i noisefilename -o outputfilename inputfilename long integer sound samples
  -f -i noisefilename -o outputfilename inputfilename float sound samples
  -V -i noisefilename -o outputfilename inputfilename verbose status information
  -R -i noisefilename -o outputfilename inputfilename continually rewrite header while writing soundfile
  -- filename -i noisefilename -o outputfilename inputfilename output to log file filename
  -H1 -i noisefilename -o outputfilename inputfilename generates a rotating line progress report
  -H2 -i noisefilename -o outputfilename inputfilename generates a "." every time a buffer is written
  -H3 -i noisefilename -o outputfilename inputfilename reports the size in seconds of the output
  -H4 -i noisefilename -o outputfilename inputfilename sounds a bell for every buffer of the output written
 
srconv -o filename inputfilename output filename
  -r num inputfilename output sample rate
  -P num inputfilename pitch transposition ratio
  -Q num inputfilename quality factor in the range of 1 to 4
  -i filename inputfilename break file
  -A inputfilename AIFF format output
  -W inputfilename WAV format output
  -J inputfilename IRCAM format output
  -h inputfilename skip soundfile header
  -8 inputfilename 8-bit unsigned character sound samples
  -c inputfilename 8-bit signed character sound samples
  -a inputfilename alaw sound samples
  -u inputfilename ulaw sound samples
  -s inputfilename short integer sound samples
  -l inputfilename long integer sound samples
  -f inputfilename float sound samples
  -K inputfilename do not generate PEAK chunks
  -R inputfilename continually rewrite header while writing soundfile
  -H1 inputfilename generates a rotating line progress report
  -H2 inputfilename generates a "." every time a buffer is written
  -H3 inputfilename reports the size in seconds of the output
  -N inputfilename sounds a bell for every buffer of the output written
  -- filename inputfilename output to log file filename
