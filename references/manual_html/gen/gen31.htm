GEN Routines: Sine/Cosine Generators

GEN31

  f  #    time    size   31   src  pna stra phsa  pnb strb phsb  . . .

Description

This routine is similar to GEN09, but allows mixing any waveform specified in an existing table.

Initialization

src – source table number

pna, pnb, etc. – partial number, must be a positive integer

stra, strb, etc. – amplitude scale

phsa, phsb, etc. – start phase (0 to 1)

Author

Istvan Varga
2001
