GEN Routines: Waveshaping

GEN30

  f  #    time    size   30   src  minh  maxh  [ref_sr]

Description

GEN30 extracts a range of harmonic partials from an existing waveform.

Initialization

src – source ftable

minh – lowest harmonic number

maxh – highest harmonic number

ref_sr (optional) – maxh is scaled by (sr / ref_sr). Default value is sr.

Author

Istvan Varga
2001
New in Csound 4.16
