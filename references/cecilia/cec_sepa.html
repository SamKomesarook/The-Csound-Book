NAME

csepar - Creation of a separator in the list of graphs

SYNOPSIS

csepar   ?name?

DESCRIPTION

Inserts an empty  line between different graph names in the interface. If a ?name? is given, the name replaces the empty line. Objects are created in the order they are declared.
