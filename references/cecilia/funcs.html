Cybil functions

A Cybil function generates one value each time it is called. Functions cannot be used to generate multiple pieces of data. To do this, they must be embedded into an algorithm. The legal Cybil functions are: 
description  function     syntax
___________       __      _________________________________

random          ran     {ran f/i min max}
randompick      pik     {pik val1 val2 val3 val3 val4....}
parameter call  pa      {pa #}
constant        co      {co #}
