The co function

{co number}
co is for constant. co will generate a constant. Functions have no duration and can only be used inside an algorith or as an operand.

number can be:

  1. a real number. 
    {co 1.2} 
{co 6000}
  2. a CECILIA irate variable. Any irate variable from the CECILIA interface can be used (sliders, toggles, option) by including the name of the variable between [] brackets. The substitution is done before the score is pre-compiled. 
    {co [blue]}
    where blue has been defined as irate variables in the CECILIA interface.
