The gr algorithm

{gr name_of_graph duration(s)}
The gr is for graph. The name given after gr will indicate a previously defined graph object in the CECILIA interface to be used as a map for values to be generated. Last element is the duration in number of events or seconds (s).

Name can only be a previously defined graph from the interface: 

{gr blueshoes 10s}
