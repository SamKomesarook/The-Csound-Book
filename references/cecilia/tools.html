USING THE  GRAPHING TOOLS

General
Description
 
 

General  top

The CECILIA interface window offers some very powerful tools to generate and modify functions. These are accessed with the row of buttons under the menubar in the interface window. The first set of tools        let you copy, paste, reset and delete functions as well as writing them to/from file.

THe other tools let you generate and modify functions in many ways. Hence, these all have parameters boxes associated with them. These parameter boxes will open up when selected with Button-2 or with Shift-Button-1. Relevant parameters can be entered with checkbuttons and sliders.
