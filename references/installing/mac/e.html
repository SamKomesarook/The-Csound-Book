Steps 17-20

 

17. Click select to choose a Csound piece.

 

18. Pick an .orc file. The.orc file is generally pair with a .sco file with the same name in the same folder. If the .sco is not in the same folder, the .sco file will not be automatically selected.

 

19. To compile the .orc and .sco, click render. Some files take no more than a couple of seconds while others can take much longer. Time varies with speed of computer. Be patient.

 

20. Once the file is rendered, use the transport to play and stop the soundfile.
