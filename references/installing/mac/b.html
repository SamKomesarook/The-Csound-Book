Selecting a Text Editor

5. When Csound is opened for the first time, a window pops up to inform you that 'No Text Editor has been selected!'. To select a text editor, click select. If you wish to do this later, click cancel and skip to Step #7.

 

6. Locate BBEdit or any text editor of your choice on you drive. Once the file has been located, highlight the application and click open.
