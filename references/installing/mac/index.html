Welcome to Csound! In these 20 short easy steps, you will be able to install, setup, and run Csound.

 

Install Csound

1. Open Csound file by double clicking on the icon.

 

2. Once the contents of the file is revealed, drag contents on to desktop or to any folder of your choice.

 

3. Double Click Csound Folder to open.

 

4. To launch Csound, double click on the Csound applications.
