Editorial

Hans Mikelson

hljmm@werewolf.net

Hello and welcome to the magazine. If you are a new visitor I think you will find a refreshing dose of content, code and examples that is rare on the web. If you are a returning visitor you should find more of what you have found in the past. I have changed the format to use tables instead of frames. This should help with printing the articles a little. I also put all of the magazine article links in a row on the left which should make navigation a little less confusing.

The beginners column covers envelopes this issue. There was some extensive discussion of delay on the Csound mailing list lately which inspired me to write an article on using delay for the effects column. For the real-time section I describe how to use Hubi's loopback device to play Csound in real-time and record to a Midi sequencer at the same time. The synthesis column describes using the Julia set as a sound source.

I seem to be able to create instruments easily but I always have trouble composing so I am happy to welcome Michael Rhoades with his insights on composition. Have you ever wanted to embed Csound on a web page? Check out Michael Gogins article on AXCsound. Damián Keller offers a tutorial on using CMask for ecological modeling. Finally Dennis Miller was kind enough to contribute a cover image for this issue. The cover image was created from the quaternion julia set and rendered using POVRay.

Enjoy,

Hans Mikelson

Csound Magazine Editor
