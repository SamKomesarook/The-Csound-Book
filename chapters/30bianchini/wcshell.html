WCShell

version 3.16

A Windows 95 (tm) shareware package for Csound users.

WCShell is a command console to Csound, includes orchestra & score editors, can run up to 4 different versions of Csound, has an Option Menu from which you can run up to 10 programs. It is not YACS (Yet Another Csounder). Its main options are configurable by user.

It includes other software, among which:

  • Scorex, a score processor; your score will be transformed in a spreadsheet, with options for modifying a single cell or a block of cells. Among the modifications: Add, Multiply, Linear Interpolate, Exp Interpolate, Fill, Convert from/to oppc, opd, cps. A logical editor (that resembles Cubase logical editor) allows you to select non-adjacent notes.
  • Mask, a tendency mask score generator.
  • GraphSco, a score generator from a bitmap image.
  • WavFFt, (sorry, still a MS-DOS app... Windows version coming soon!) to analyze wave files both in valleys/mountains and spectrogram styles.
  • Harmonize, a score modifier. It adds to a monodic score "chords" selected among a given repertoir.
  • Grafica, a score viewer in Stockhausen's Studie II style (time vs. pitch/amp).
  • Retro, to retrogradize a score.
  • EditHet, to graphically edit a hetro  generated file

Recommended: MIDI Realtime Csound for Windows95

(by Gabriel Maldonado, http://www.agora.stm.it/G.Maldonado.htm).

WCShell features:

  • Text boxes for current (or selected) orchestra, score and audio file.
  • A multimedia control
  • A RUN button, to fast execute last version of Csound with unchanged parameters
  • Three drive/dir/file boxes, for orc, sco and wave, with Update buttons.
  • Orc menu options:
    • Edit: edit current orchestra (shortcut F1)
    • Custom Editor: edit current orchestra with custom editor
    • New: open a new orchestra
    • View Comments: display in a windows only commented lines
    • Macro Processor: create a new orchestra with expanded macros
  • Sco menu options:
    • Edit: edit current score (shortcut F2)
    • Custom Editor: edit current score with custom editor
    • New: open a new score
    • Grafica: display score in graphic style
    • Draw Parameters: graphically modify/add parameters to an existing score
    • MidiToScore: convert a Standard Midi File in Csound score
    • ScoreToMidi: convert a Csound score in Standard Midi File
    • Macro Processor: create a new orchestra with expanded macros
  • Wave menu options:
    • Play: play current audio file (shortcut F3)
    • View Wave: display wave file in blue zone
    • Sndinfo: wave file info (shortcut F7)
    • Peak. measures positive and negative max peaks
    • Transpose: transpose a wave file by linear interpolation
    • Recalc Header: recalc header data of a wave file (i.e. when a Csound synthesis is interrupted)
    • Wave Editor: run selected wave file editor
  • Run menu options:
    • Run selected version of Csound, with or without a dialog box appears
  • View menu options:
    • Show Dir, Drive/Hide Dir, Drive to show/hide Dir/Drive boxes
  • Util menu options:
    • Files info: display info (date, time, size) about current orc, sco, wave files
    • DOS Prompt
    • Kill file
    • WavFft
    • Revert to saved: WCShell dos not allow you to run a synthesis unless you saved last modifications to orc & sco files. Revert to Saved overrides this security feature.
    • Text Editor: run a simple text editor
    • Merge scores: orchestra & score editors cannot open big files. So you can edit a number of smaller files, and merge them before running Csound.
    • Mixer: run selected audio board mixer program
    • Composition:
      • Mask: run Mask program
      • Score Processor: run Scorex program with selected score (shortcut F9)
      • GraphSco: run GraphSco program
    • Analysis:
      • Hetro: run hetro program
      • EditHetro: run EditHet program
      • PVAnal: run pvanal program
      • CVAnal. run cvanal program
      • LPCAnal: run lpanal program
    • Fonts: choose font type & size for editors
    • Settings: view/modify settings: wave file editor, mixer program, option menu voices etc.
  • Window menu options:
    • Strip Windows: open orc & sco editors and places them horizontally (shortcut SHIFT-F1)
    • Tile Windows: open orc & sco editors and places them vertically (shortcut SHIFT-F2)
    • Close Editors: closes orc & sco editors (shortcut SHIFT-F3)
  • Options menu: configurable (see Settings) menu to run up to 10 programs
  • Exit options menu:
    • Yes/No: confirm exit
  • Help options menu:
    • Help: WCShell online help
    • Manual: Csound online help
    • Info: info on current WCshell version

WCShell is written in Microsoft Visual Basic 4.0. It needs Windows95 or WindowsNT, at least 6 Mbyte of free disk space.
