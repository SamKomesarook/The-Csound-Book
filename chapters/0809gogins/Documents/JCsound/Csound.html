run

public final void run()
Runs Csound with the current command line. The standard Csound messages are printed to the Java console. The rendering thread can be suspended, resumed, and stopped, and a new rendering thread can then be started without reloading the Csound shared library. Note that although Csound runs in a separate thread, it is not possible for a process to run more than one Csound thread at the same time.
