JCsound provides a Java interface to Csound, a freely available software synthesizer that is available on many computer platforms.

JCsound.Csound has low-level functions that are implemented in native code by JCsound_Csound, a shared library version of Csound that is re-entrant (it can be stopped and started again without reloading).

JCsound.CsoundManager provides both a graphical user interface and a higher-level programmatic interface for controlling Csound and for manipulating Csound files.

JCsound.CsoundManager is used as the software synthesizer for Silence, an extensible system for making music on computers by means of software alone. JCsound can be used as a synthesizer component in other software, such as score generatoring programs. It can also run as a standalone "launcher" for Csound.
