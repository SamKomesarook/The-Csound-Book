subsub

C-language program (to accompany tract/intract Csound orchestra file)
©1996 by Noel Bush

Please note: there may be small differences between the HTML-ized versions of these files and the actual source (located in the src-text directory). Please see the author's website at http://www.erols.com/noelbush for the latest updates.

defg.h: general macros, typedefs, prototypes

defd.h: macros for pre-sco operations

defs.h: macros for sco operations

cal.c: value converters, special functions

  • ampdb(): convert decibels to raw amplitude
  • avg(): find an average
  • betw(): hard limit a value to a range
  • cpsoct(): convert octave.decimal notation to Hertz
  • dbamp(): convert raw amplitude to decibels
  • fbetw(): limit a value to a range by repeatedly halving or doubling it
  • getmaxmin(): get the maximum and minimum values from a datafield
  • gof(): return the greater of two inputs
  • linscale(): linearly scale a value to fall in a destination range proportionally to its location in an original range
  • lof(): return the lesser of two inputs
  • logscale(): logarithmically scale a value to fall in a destination range proportionally to its location in an original range
  • octcps(): convert Hertz to octave.decimal notation
  • pc(): round off an octave.decimal value to the nearest pitch-class (within a specified number of pitch-classes)
  • quant(): quantize a value to the nearest specified increment
  • susum(): find the "supersum" of a number

choix.c: handle choices, adjustments, data entry

  • adj_params(): adjust the global parameters of the subsub project
  • check_pause(): check for screen height break and pause for keypress
  • enteredit_specxy(): enter and edit minspec datafield
  • get_filename(): input a filename
  • getack(): get acknowledgement
  • task(): subsub main menu
  • wh(): choice entry function

data.c: make pre-scorefile datafields

  • calc_hd_hr(): calculate hd and hr factors
  • copy_specxy(): make a specplex from the minspec
  • make_fullplex(): create fullplex by scaling specplex
  • s_and_q(): scale and quantize fullplex

instr_1.c: instrument 1 ("raw") - see orchestra

instr_2.c: instrument 2 ("processed #1") - see orchestra

instr_4.c: instrument 4 ("processed #2") - see orchestra

main.c: main function

par.c: initializations

sar.c: store and retrieve datafields

  • load()
  • readlist()
  • readplex()
  • tryopen()
  • write()
  • writelist()
  • writeplex()

sco.c: make scorefile

  • comp(): simple comparison function
  • make_sco(): main scoremaking function
  • talmax(): return most-occuring pitch-class as of tally-moment
  • talmid(): return middlemost-occuring pitch-class as of tally-moment
  • talmin(): return least-occuring pitch-class as of tally-moment

see.c: display datafields

  • display()
  • show_fullplex()
  • show_minspec()
  • show_params()
  • show_sco()
  • show_specplex()
